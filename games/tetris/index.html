About This Game

Classic falling-block puzzle action — guide seven different tetromino shapes as they drop, rotate them to fit, and clear complete horizontal lines before the stack reaches the top. Speed increases every 10 lines, so the challenge never stops.

Controls

← →  Move piece left / right
↑  Rotate clockwise
Z  Rotate counter-clockwise
↓  Soft drop (faster fall)
Space  Hard drop (instant lock)
P  Pause / resume
R  Restart after game over

On mobile: use the ← → buttons to move, ↺ to rotate, and ↓ to drop. Tapping the screen also starts the game and restarts after game over.
